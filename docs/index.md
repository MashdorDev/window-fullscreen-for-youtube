# Window Fullscreen for YouTube

Watch YouTube in true **windowed fullscreen** — fill your entire browser window,
edge to edge, without switching the operating system into native fullscreen. Your
taskbar, tabs, and other windows stay reachable.

Free, open source (MIT), no ads, no tracking, no data collection.

## What it does

The extension adds a windowed-fullscreen button to YouTube's player controls (and a
configurable hotkey, default `Shift+F`). On top of that single mode it offers a few
optional refinements: auto-entering on each new video, a scrollable layout that keeps
comments and related videos reachable, a pinned live-chat side panel, and granular
control over which page elements hide while active.

It runs only on `youtube.com` and does nothing on any other site.
